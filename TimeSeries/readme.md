# Time Series Analysis
Structure of Time Series analysis
* Trend - the general direction of the data
* Seasonality - cyclic patterns in the data
* Exogenous Regressors - external events that can affect the analysis
* Error Term - what can not be explained by the above 3 factors (error is part of the prediction)<br>
